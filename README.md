# ChatSample

Demo project utilizing technologies like:
  * React + TypeScript
  * Net Core 3.1
  * SignalR
  * Google login
  * EF Core 3.1
  * React hooks + custom hooks
  * Styled-components
  * React Transition Group (for animations)
